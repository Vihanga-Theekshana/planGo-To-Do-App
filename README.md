# 🚀 PlanGo - To-Do Android App

PlanGo is a simple and modern To-Do application built using Android Studio.  
It helps users manage daily tasks easily and stay productive.

---

## ✨ Features

- ➕ Add new tasks  
- 🗑️ Delete tasks  
- ✅ Mark tasks as completed  
- 📋 Simple and clean user interface  
- 💾 Lightweight and fast  

---

## 🛠️ Built With

- Android Studio  
- Java  
- XML (UI Design)
